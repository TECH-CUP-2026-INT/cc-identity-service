# Team

This documentation site is maintained by the **TECH-CUP 2026 INT** team as part of the DOSW platform initiative of the Escuela Colombiana de Ingeniería Julio Garavito.

## Contributors

The CC Identity Service is developed by the authentication squad within TECH-CUP 2026 INT. Team members and roles are tracked in the project's GitHub repository.

## How to contribute

1. Open an issue describing the change.
2. Create a feature branch from `main` (e.g. `feat/your-name`).
3. Keep the documentation in `docs/` in sync with code changes — docs live on the `feature-Rojas` branch for review.
4. Open a pull request; CI builds the MkDocs site before merge.
